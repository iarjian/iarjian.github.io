yarn和npm常用指令

2019-11-07
npm yarn 用法
npm init yarn init 初始化项目
npm install yarn / yarn install 一键安装package.json的所有包
npm install [package] --save / -S yarn add [package] 安装到dependencies , 默认保存到package
npm install [package] --save-dev/ -D yarn add [package] --save-dev/ -D 安装到devDependencies(开发时的依赖项目)
npm i [package]@[版本号] yarn add [package]@[版本号] 指定版本号下载更新
npm install [package] --global yarn global add [package] 全局安装某个包
npm update --global yarn updade upgrade 更新所有包
npm uninstall [package] yarn remove [package] 删除某个包
npm publish/login/loginout yarn publish/login/loginout 发布/登录/登出,一系列NPM Registry操作
npm run/test yarn run/test 运行某个命令
